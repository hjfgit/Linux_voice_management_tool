# Linux_voice_management_tool
# 项目简介
1、使用C++编写一个智能AI对话和语音命令执行的语音管理工具

2、可执行命令支持配置

# 使用技术
1、C++STL

2、http第三方库

3、图灵机器人

4、百度语音识别和语音合成

5、Linux系统知识 

#使用指南

1、从github克隆到本地：
    git clone https://github.com/hjfgit/Linux_voice_management_tool.git

2、进入到Linux_voice_management_tool目录：
    cd Linux_voice_management_tool

3、make进行编译
    make

4、./Jarvis 进行执行
    ./Jarvis
